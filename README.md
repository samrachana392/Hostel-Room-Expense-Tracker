# Hostel/Room Expense Tracker
A simple **Python console application** to manage and track monthly expenses for hostels or shared living spaces. Split bills among roommates and keep a record of past expenses easily.

## Features
- Add monthly expenses including:
  - Rent
  - Food
  - Water
  - Electricity (units Ã— charge per unit)
  - Other miscellaneous costs
- Calculate total expense and per-person share
- View complete expense history
- Clear all expense history when needed   
- Easy-to-use menu-driven interface
